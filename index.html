WWUD-CS

*WWUD-CS *is an ASP.net MVC5 site written in C# using Entity Framework 6. I'm using this site as a testing ground for me to learn asp.net and mvc technologies.

The Development branch is my working branch and will change constantly. It will always contain the latest code but may be broken at any given time.

The Master branch will be updated periodically from development.
